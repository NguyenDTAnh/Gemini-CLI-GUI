# Gemini CLI Chat Extension

VS Code extension with an interactive chat UI backed by local Gemini CLI.

## Quick start

1. Install dependencies:

   npm run bootstrap

2. Build webview + extension:

   npm run build

3. Open the workspace in VS Code and press F5 to run Extension Development Host.

## Workspace scripts

- npm run bootstrap
- npm run build
- npm run build:webview
- npm run build:extension
- npm run watch:webview
- npm run watch:extension
- npm run package:vsix
